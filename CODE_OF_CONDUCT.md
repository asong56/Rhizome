# Code of Conduct

## Our Pledge

Rhizome is a network built on the principle of peer connection — no hierarchy, no gatekeeping. We extend that same spirit to how we interact with each other. Everyone who participates in this project, whether by joining the network, opening an issue, or contributing code, is expected to make participation a harassment-free experience for everyone, regardless of age, body size, disability, ethnicity, gender identity and expression, level of experience, nationality, personal appearance, race, religion, or sexual identity and orientation.

## Our Standards

**Behavior that contributes to a positive environment:**

- Being respectful and considerate in all communications
- Welcoming different perspectives, aesthetics, and approaches to the indie web
- Giving and gracefully accepting constructive feedback
- Focusing on what is best for the network and its members
- Showing empathy toward other community members

**Behavior that is not acceptable:**

- Harassment, intimidation, or discrimination of any kind
- Trolling, insulting or derogatory comments, and personal or political attacks
- Publishing others' private information without explicit permission
- Submitting nodes that host illegal content, malware, spam, or hate speech
- Any other conduct which could reasonably be considered inappropriate

## Scope

This Code of Conduct applies to all project spaces — GitHub issues, pull requests, discussions, and any other forum associated with this project. It also applies when an individual is representing the project or its community in public spaces.

## Enforcement

Instances of abusive, harassing, or otherwise unacceptable behavior may be reported by opening a GitHub issue or contacting the maintainer directly. All complaints will be reviewed and investigated promptly and fairly.

The maintainer reserves the right to remove any node from `nodes.json`, close any issue, or block any account that violates this Code of Conduct.
